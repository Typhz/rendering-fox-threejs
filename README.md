## 🚀 Como executar o projeto

#### Clone o repositório
```bash
git clone https://github.com/Typhz/rendering-fox-threejs.git
```

#### Instale as dependências
```bash
cd <REPO_NAME>
yarn install
```

#### Rode a aplicação
```bash
yarn dev
```
